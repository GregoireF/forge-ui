**Tags Input** lets users type values and convert them into removable tag chips. Pressing `Enter` or `,` adds a tag; clicking the delete trigger or pressing `Backspace` removes one.

## Usage

<Tabs>
  <TabItem label="React">
    ```tsx
    import { TagsInput } from "@forge-ui/react";

    export function SkillsTags() {
      return (
        <TagsInput.Root>
          {({ tags }) =>
            tags.map((tag) => (
              <TagsInput.Tag key={tag.id} tag={tag}>
                <TagsInput.TagText>{tag.value}</TagsInput.TagText>
                <TagsInput.TagDelete tag={tag}>×</TagsInput.TagDelete>
              </TagsInput.Tag>
            ))
          }
          <TagsInput.Input placeholder="Add skill..." />
          <TagsInput.HiddenInput />
        </TagsInput.Root>
      );
    }
    ```
  </TabItem>
  <TabItem label="Vue">
    ```vue
    <script setup lang="ts">
    import {
      TagsInputRoot, TagsInputTag, TagsInputTagText,
      TagsInputTagDelete, TagsInputInput, TagsInputHiddenInput,
    } from "@forge-ui/vue";
    </script>

    <template>
      <TagsInputRoot v-slot="{ tags }">
        <TagsInputTag v-for="tag in tags" :key="tag.id" :tag="tag">
          <TagsInputTagText>{{ tag.value }}</TagsInputTagText>
          <TagsInputTagDelete :tag="tag">×</TagsInputTagDelete>
        </TagsInputTag>
        <TagsInputInput placeholder="Add skill..." />
        <TagsInputHiddenInput />
      </TagsInputRoot>
    </template>
    ```
  </TabItem>
</Tabs>

## Validation

```tsx
// Prevent duplicates
<TagsInput.Root validate={(value, tags) => !tags.some((t) => t.value === value)}>

// Max tags
<TagsInput.Root max={5}>

// Custom delimiter
<TagsInput.Root delimiter={[",", " "]}>
```

## API Reference

### TagsInput.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string[]` | — | Controlled tags array |
| `defaultValue` | `string[]` | `[]` | Uncontrolled initial tags |
| `onValueChange` | `(tags: string[]) => void` | — | Fires when tags change |
| `max` | `number` | — | Maximum number of tags |
| `disabled` | `boolean` | `false` | Disable all interaction |
| `readOnly` | `boolean` | `false` | Prevent tag changes |
| `delimiter` | `string \| string[]` | `[","]` | Characters that trigger tag creation |
| `validate` | `(value: string, tags: Tag[]) => boolean` | — | Custom validation per tag |
| `name` | `string` | — | Hidden input field name |

### TagsInput.Tag

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `tag` | `Tag` | ✅ | Tag object from the render slot |

### TagsInput.TagDelete

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `tag` | `Tag` | ✅ | Tag object to delete |

## Data attributes

| Attribute | Element | Values |
|-----------|---------|--------|
| `data-forge-scope` | All parts | `"tags-input"` |
| `data-forge-part` | All parts | `"root"` \| `"tag"` \| `"tag-text"` \| `"tag-delete-trigger"` \| `"input"` \| `"hidden-input"` \| `"label"` |
| `data-focused` | `root` | `"true"` when input is focused |
| `data-disabled` | `root`, `tag` | `"true"` |
| `data-readonly` | `root` | `"true"` |
| `data-value` | `tag` | The tag's string value |

## Accessibility

- Root has `role="group"`; each tag has `role="row"` with its text and delete button
- `aria-disabled` propagates from Root to prevent interaction when disabled
- `aria-readonly` propagates for read-only mode

## Keyboard interactions

| Key | Action |
|-----|--------|
| `Enter` | Add current input as tag |
| `,` (or custom delimiter) | Add current input as tag |
| `Backspace` | Remove last tag when input is empty |
| `←` / `→` | Navigate between tags when input is empty |
| `Delete` | Remove currently focused tag |
