A **Radio Group** presents a set of mutually exclusive choices. Only one item can be checked at a time. Navigation uses roving tabIndex so the group behaves as a single tab stop.

## Usage

<Tabs>
  <TabItem label="React">
    ```tsx
    import { RadioGroup } from "@forge-ui/react";

    export function PlanSelector() {
      return (
        <RadioGroup.Root defaultValue="pro">
          {["free", "pro", "enterprise"].map((plan) => (
            <RadioGroup.Item key={plan} value={plan}>
              <RadioGroup.Radio value={plan} />
              <RadioGroup.Label value={plan}>{plan}</RadioGroup.Label>
            </RadioGroup.Item>
          ))}
          <RadioGroup.HiddenInput name="plan" />
        </RadioGroup.Root>
      );
    }
    ```
  </TabItem>
  <TabItem label="Vue">
    ```vue
    <script setup lang="ts">
    import {
      RadioGroupRoot, RadioGroupItem, RadioGroupRadio,
      RadioGroupLabel, RadioGroupHiddenInput,
    } from "@forge-ui/vue";
    </script>

    <template>
      <RadioGroupRoot default-value="pro">
        <RadioGroupItem v-for="plan in plans" :key="plan" :value="plan">
          <RadioGroupRadio :value="plan" />
          <RadioGroupLabel :value="plan">{{ plan }}</RadioGroupLabel>
        </RadioGroupItem>
        <RadioGroupHiddenInput name="plan" />
      </RadioGroupRoot>
    </template>
    ```
  </TabItem>
</Tabs>

## API Reference

### RadioGroup.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected value |
| `defaultValue` | `string` | — | Uncontrolled initial value |
| `disabled` | `boolean` | `false` | Disable the entire group |
| `required` | `boolean` | `false` | Mark as required |
| `name` | `string` | — | Form field name shared across items |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | Affects arrow key navigation |
| `onValueChange` | `(value: string) => void` | — | Fires on selection change |

## Data attributes

| Attribute | Element | Values |
|-----------|---------|--------|
| `data-forge-scope` | All parts | `"radio-group"` |
| `data-forge-part` | All parts | `"root"` \| `"item"` \| `"radio"` \| `"label"` |
| `data-state` | `item`, `radio` | `"checked"` \| `"unchecked"` |
| `data-disabled` | All parts | `""` |

## Keyboard interactions

| Key | Action |
|-----|--------|
| `ArrowDown` / `ArrowRight` | Select next item |
| `ArrowUp` / `ArrowLeft` | Select previous item |
| `Space` | Select focused item |
