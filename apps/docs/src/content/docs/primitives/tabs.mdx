**Tabs** organize content into panels that are revealed one at a time. Supports automatic (focus) or manual (click) activation modes, horizontal and vertical orientation, and roving tabIndex keyboard navigation.

## Usage

<Tabs>
  <TabItem label="React">
    ```tsx
    import { Tabs } from "@forge-ui/react";

    export function MyTabs() {
      return (
        <Tabs.Root defaultValue="tab1">
          <Tabs.List>
            <Tabs.Trigger value="tab1">Overview</Tabs.Trigger>
            <Tabs.Trigger value="tab2">Details</Tabs.Trigger>
          </Tabs.List>
          <Tabs.Panel value="tab1">Overview content</Tabs.Panel>
          <Tabs.Panel value="tab2">Details content</Tabs.Panel>
        </Tabs.Root>
      );
    }
    ```
  </TabItem>
  <TabItem label="Vue">
    ```vue
    <script setup lang="ts">
    import { TabsRoot, TabsList, TabsTrigger, TabsPanel } from "@forge-ui/vue";
    </script>

    <template>
      <TabsRoot default-value="tab1">
        <TabsList>
          <TabsTrigger value="tab1">Overview</TabsTrigger>
          <TabsTrigger value="tab2">Details</TabsTrigger>
        </TabsList>
        <TabsPanel value="tab1">Overview content</TabsPanel>
        <TabsPanel value="tab2">Details content</TabsPanel>
      </TabsRoot>
    </template>
    ```
  </TabItem>
</Tabs>

## Controlled

```tsx
const [tab, setTab] = useState("overview");
<Tabs.Root value={tab} onValueChange={setTab}>...</Tabs.Root>
```

## API Reference

### Tabs.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled active tab |
| `defaultValue` | `string` | — | Uncontrolled initial tab |
| `activationMode` | `"automatic" \| "manual"` | `"automatic"` | Auto-select on focus or require click |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Direction of the tablist |
| `onValueChange` | `(value: string) => void` | — | Fires on tab change |

### Tabs.Trigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | **required** | The tab value this trigger controls |
| `disabled` | `boolean` | `false` | Prevent activation of this tab |

### Tabs.Panel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | **required** | Must match the corresponding trigger value |
| `forceMount` | `boolean` | `false` | Keep in DOM when inactive (for exit animations) |

## Accessibility

- Tablist has `role="tablist"` and `aria-orientation`
- Each trigger has `role="tab"`, `aria-selected`, `aria-controls`
- Each panel has `role="tabpanel"` and `aria-labelledby`
- Follows the [WAI-ARIA Tabs pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tabs/)

## Data attributes

| Attribute | Element | Values |
|-----------|---------|--------|
| `data-forge-scope` | All parts | `"tabs"` |
| `data-forge-part` | All parts | `"root"` \| `"list"` \| `"trigger"` \| `"panel"` |
| `data-state` | `trigger`, `panel` | `"active"` \| `"inactive"` |
| `data-orientation` | `root`, `list`, `trigger` | `"horizontal"` \| `"vertical"` |

## Keyboard interactions

| Key | Action |
|-----|--------|
| `ArrowRight` / `ArrowDown` | Move focus to next tab |
| `ArrowLeft` / `ArrowUp` | Move focus to previous tab |
| `Home` | Move focus to first tab |
| `End` | Move focus to last tab |
| `Enter` / `Space` | Activate focused tab (manual mode) |
