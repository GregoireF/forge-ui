A **Slider** is an input that lets users select a value from a continuous or stepped range. It supports pointer drag, keyboard navigation, custom min/max/step, and renders `role="slider"` with full ARIA attributes.

## Usage

<Tabs>
  <TabItem label="React">
    ```tsx
    import { Slider } from "@forge-ui/react";

    export function VolumeSlider() {
      return (
        <Slider.Root defaultValue={50} min={0} max={100}>
          <Slider.Track>
            <Slider.Range />
          </Slider.Track>
          <Slider.Thumb aria-label="Volume" />
          <Slider.HiddenInput name="volume" />
        </Slider.Root>
      );
    }
    ```
  </TabItem>
  <TabItem label="Vue">
    ```vue

      <SliderRoot :default-value="50" :min="0" :max="100">
        <SliderTrack>
          <SliderRange />
        </SliderTrack>
        <SliderThumb aria-label="Volume" />
      </SliderRoot>
    </template>
    ```

  </TabItem>

## Controlled

```tsx
const [volume, setVolume] = useState(50);

<Slider.Root value={volume} onValueChange={setVolume}>
  ...
</Slider.Root>
```

## Custom step

```tsx
<Slider.Root defaultValue={0} min={0} max={1} step={0.01}>
  <Slider.Track><Slider.Range /></Slider.Track>
  <Slider.Thumb />
</Slider.Root>
```

## Vertical orientation

```tsx
<Slider.Root orientation="vertical" defaultValue={50}>
  <Slider.Track>
    <Slider.Range />
  </Slider.Track>
  <Slider.Thumb />
</Slider.Root>
```

## API Reference

### Slider.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Controlled value |
| `defaultValue` | `number` | `min` | Uncontrolled initial value |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Slider direction |
| `disabled` | `boolean` | `false` | Disable interaction |
| `onValueChange` | `(value: number) => void` | — | Fires on every change |
| `onValueCommit` | `(value: number) => void` | — | Fires on pointerup / keyboard commit |

## Data attributes

| Attribute | Element | Values |
|-----------|---------|--------|
| `data-forge-scope` | All parts | `"slider"` |
| `data-forge-part` | All parts | `"root"` \| `"track"` \| `"range"` \| `"thumb"` |
| `data-orientation` | All parts | `"horizontal"` \| `"vertical"` |
| `data-disabled` | All parts | `""` |

## Accessibility

- Thumb has `role="slider"`, `aria-valuenow`, `aria-valuemin`, `aria-valuemax`, `aria-valuetext`, `aria-orientation`
- `aria-disabled` propagates when the slider is disabled
- Follows the [WAI-ARIA Slider pattern](https://www.w3.org/WAI/ARIA/apg/patterns/slider/)

## Keyboard interactions

| Key | Action |
|-----|--------|
| `ArrowRight` / `ArrowUp` | Increment by step |
| `ArrowLeft` / `ArrowDown` | Decrement by step |
| `PageUp` | Increment by 10% of range |
| `PageDown` | Decrement by 10% of range |
| `Home` | Set to minimum |
| `End` | Set to maximum |
