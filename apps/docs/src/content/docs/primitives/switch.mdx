A **Switch** is a binary toggle that represents an on/off state. It is backed by a hidden native `<input type="checkbox">` with `role="switch"`.

## Usage

<Tabs>
  <TabItem label="React">
    ```tsx
    import { Switch } from "@forge-ui/react";

    export function NotificationSwitch() {
      return (
        <Switch.Root>
          <Switch.Control>
            <Switch.Thumb />
          </Switch.Control>
          <Switch.Label>Email notifications</Switch.Label>
          <Switch.HiddenInput />
        </Switch.Root>
      );
    }
    ```
  </TabItem>
  <TabItem label="Vue">
    ```vue
    <script setup lang="ts">
    import {
      SwitchRoot, SwitchControl, SwitchThumb, SwitchLabel,
    } from "@forge-ui/vue";
    </script>

    <template>
      <SwitchRoot>
        <SwitchControl>
          <SwitchThumb />
        </SwitchControl>
        <SwitchLabel>Email notifications</SwitchLabel>
      </SwitchRoot>
    </template>
    ```
  </TabItem>
</Tabs>

## Controlled state

<Tabs>
  <TabItem label="React">

    ```tsx
    </Switch.Root>
    ```

## API Reference

### Switch.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Uncontrolled initial state |
| `onCheckedChange` | `(checked: boolean) => void` | — | Fires on state change |
| `disabled` | `boolean` | `false` | Disable the switch |
| `required` | `boolean` | `false` | Mark as required |
| `name` | `string` | — | Form field name |
| `value` | `string` | `"on"` | Form field value |

## Data attributes

| Attribute | Element | Values |
|-----------|---------|--------|
| `data-forge-scope` | All parts | `"switch"` |
| `data-forge-part` | All parts | `"root"` \| `"control"` \| `"thumb"` \| `"label"` \| `"hidden-input"` |
| `data-state` | `root`, `control` | `"checked"` \| `"unchecked"` |
| `data-disabled` | `root`, `control` | `"true"` |

## Keyboard interactions

| Key | Action |
|-----|--------|
| `Space` | Toggle checked state |
