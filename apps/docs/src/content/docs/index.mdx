## Why forge-ui?

forge-ui is a headless component library that owns the **behaviour** and **accessibility** of UI primitives so you own the **look**. It ships zero styles. Instead it exposes a predictable CSS contract via `data-forge-scope` and `data-forge-part` attributes — the same API across React and Vue.

<CardGrid>
  <Card title="Headless" icon="puzzle">
    Zero default styles. Ship your design system, not ours.
  </Card>
  <Card title="CSS contract" icon="pencil">
    Target every part via `[data-forge-scope="dialog"][data-forge-part="overlay"]`. No class name coupling.
  </Card>
  <Card title="React & Vue" icon="laptop">

  </Card>
  <Card title="Accessible" icon="approve-check-circle">

  </Card>
  <Card title="3-tier architecture" icon="setting">

  </Card>
  <Card title="Tested" icon="magnifier">
    Every primitive has unit and interaction tests for both React and Vue.
  </Card>
</CardGrid>
